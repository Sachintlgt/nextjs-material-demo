## Getting Started

Installation
```bash
npm i
```

Create a new file .env
Copy .env.sample file into .env

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Testing
```bash
npm run test
```
